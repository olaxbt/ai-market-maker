# Security Policy

## Reporting a Vulnerability

If you find a security issue in this project, please do not open a public issue.

Report it privately through GitHub’s vulnerability reporting, or contact the maintainers directly if that is not available.

Please include:
- A clear description of the issue
- Steps to reproduce
- The affected file, branch, or commit
- Any logs, screenshots, or proof of concept

We will review valid reports as quickly as possible and follow up with next steps.
